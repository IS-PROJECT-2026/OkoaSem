# 🚀 Team Portfolio Hub

Welcome to our collective team portfolio project! We are a team of six developers/creators who have built a centralized hub showcasing our individual skills, projects, and professional journeys. 

This project is built using **Vite** for blazing-fast development and **Tailwind CSS** for modern, sleek, and responsive styling.

## 🛠️ Tech Stack & Tools

* **Bundler/Build Tool:** [Vite](https://vitejs.dev/) - Next-generation frontend tooling.
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for rapid UI development.
* **Language:** HTML5, CSS3, JavaScript (or TypeScript)
* **Icons:** [Lucide Icons](https://lucide.dev/) / [FontAwesome](https://fontawesome.com/) *(Adjust based on what you use)*

---

## 🚀 Features

* **Centralized Dashboard:** A unified landing page introducing the team and routing to individual portfolios.
* **Fully Responsive:** Optimized for mobile, tablet, and desktop screens using Tailwind's responsive utilities.
* **Dark Mode Support:** Smooth transitions between light and dark themes.
* **Performance First:** Optimized assets and minimal bundle size thanks to Vite's build pipeline.
